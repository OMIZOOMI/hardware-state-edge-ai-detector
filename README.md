## 🚀 Project Overview

This project implements a custom **YOLOv8-based computer vision pipeline** designed to detect and categorize the status of network hardware. By training on a custom-annotated dataset, the model identifies specific modem chassis and monitors LED indicator states in real-time, making it an ideal solution for remote hardware state verification.

## 📊 Dataset Engineering & Annotation
Quality data is the foundation of this model. I manually annotated frames to create a highly accurate training set.

## 🚀 How to Run
1. **Clone** the repository.
2. Open `Detecting_blackbox.ipynb` in [Google Colab](https://colab.research.google.com).
3. Update the data paths and run the cells to execute inference on your own hardware captures.
